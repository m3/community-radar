## Pain Points (by frequency)

### 1. Bug Reporting is Understaffed (71 mentions)
- **ONE person** collects ALL bug reports
- **ONE person** reproduces issues for devs
- "There simply isn't enough time in the day to personally respond to every single report"
- Bug reproduction is #1 bottleneck — live build vs dev build mismatch
- Players report bugs in main chat instead of #bug-reports channel
- Status of bug reports unknown to community

### 2. Ranked/Leaderboard Confusion (59 mentions)
- Players don't understand ranking mechanics
- Rob Bro: "I'm 15th with 33% win ratio" vs IAmLucifee: "I'm 19th with 90%"
- System weighs total wins first, then win ratio — not transparent
- Community wants ELO system but Ripstone says playerbase too small
- "It's a really compelling system - but falls apart a bit if the playerbase isn't yet big enough"

### 3. POTM Submission is Broken (59 mentions)
- Google form → no video preview
- Discord upload → 5MB limit, heavy compression
- PlayStation → 3min capture limit
- Ripstone: "Compress it into a 10mb 5fps gif and we'll try and make sense of it!"
- Mod (mmm) manually helps players upload via personal Google Drive
- POTM expanding to more platforms but still manual

### 4. Online/Lag Issues (71 mentions)
- Cross-play bug (Steam ↔ PS5) major issue June 4
- Disconnection handling affects ranked matches
- "game hangs before lag, opponent quits and I get a loss"
- Ripstone asks for: platform, region, time, opponent name, what happened

### 5. Slow Play / Deliberate Stalling (16 mentions)
- DarthThug takes 60 seconds per shot using screen cues
- "Apples" player: 40-45 seconds per shot, circumvents table twice
- Players using block function to avoid stallers

### 6. Table/Cue Customization (138 mentions — HIGHEST)

- "We build and test our table colours under specific lighting"

### 7. Xbox Version Delayed (32 mentions)

- Xbox Series S struggling: "that lack of RAM is gonna take some work"

### 8. Snooker DLC Anticipation (60 mentions)
